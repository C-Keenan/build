Game Server Helper

'Game Server Helper' is a linux command-line script interface designed to run various popular game servers such as "Minecraft", "Veloren", "Satisfactory", and "Factorio" by utilizing the 'Docker Compose' program. If you run the install script it will pull the necessary linux packages "docker-ce", "docker-ce-cli", "containerd.io", "docker-buildx-plugin", and "docker-compose-plugin". It then asks you if you would like to test the docker installation, which is not usually necessary, and run the `docker run hello-world` command, which is the 'Docker' test image, to make sure that the installation is up and running correctly.
